У каждого файла есть свой формат и структура
Формат помогает понять с помощью каких программ его стоит открывать (.mp3 - аудиопроигрыватель виндовс, .jpg - фото эдиторы и т.д.), 
а структура как прочесть этот файл (как изоброжение, видео и т.д.)

Но после определённый символов в структуре программа перестаёт читать файл и по сути в конце можно дописать секретное послание или соеденить файл с другим файлом
Чтобы создать подобные файлы надо прописать в терминале команду
cat file_1 file_2 > compare_file
Наличие склееных файлов можно проверить с помощью binwalk
binwalk file
Чтобы извлечь склееный рарник из файла можно воспользоваться binwalk
binwalk -e file

Но не все форматы могут склеиваться, а их наличие можно проверить с помощью hex редакторов
